
# nw-game-engine

A JavaScript game engine for use in making HTML/JavaScript games that compile to executables using [NW.js](http://nwjs.io/).

## Done

* Sprites
* Objects
* Keyboard input
* Gamepad input
* Game loop
* DOM utils
* Base rooms
